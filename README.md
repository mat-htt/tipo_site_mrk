#  Система управления расписанием МРК

Веб-приложение на Django для автоматизации учебного расписания.

##  Запуск
1. `pip install django`
2. `python manage.py migrate`
3. `python manage.py createsuperuser`
4. `python manage.py runserver`

##  Функционал
- Просмотр расписания с фильтром по группе.
- Добавление и редактирование записей.
- Списки преподавателей, предметов и групп.
